# MyFirstUI — WAT project scaffold

This repository is initialized as a WAT (Workflows, Agents, Tools) project scaffold per `CLAUDE.md`.

Quick start:

1. Copy `.env.example` to `.env` and fill values.
2. Run the sample tool:

```bash
python3 tools/sample_tool.py --name Alice
```

3. Add deterministic scripts to `tools/` and workflows to `workflows/`.
